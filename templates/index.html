Getting Started

Follow these steps to efficiently use the project management features:

  • Create or Select a Project: Start by creating a new project or selecting an existing one from the top bar. Once a project is selected, additional features like locations, tasks, and scheduling become available.
  • Input Locations: Use the Location page to input repeated locations for your project. These could be floors for a vertical project or road sections for a horizontal project.
  • Add Project Tasks: On the Task page, define repetitive tasks and assign logical predecessors to each task. This will help calculate timelines based on dependencies.
  • View Critical Path Attributes: Check the WBS page to view calculated CPM (Critical Path Method) attributes like Earliest Start (ES), Earliest Finish (EF), and task slack. This table helps identify which tasks are critical to the project's timeline.
  • Visualize Tasks with Gantt Charts: Use the Gantt page to visualize your schedule for tasks at a single location. This chart helps track task progress and deadlines.
  • Gantt Chart for All Locations: The Gantt Total page provides a comprehensive view of critical tasks repeated across all locations.
  • Explore Line of Balance: The Line of Balance page shows how tasks flow across all locations, ensuring that tasks start in a sequence that avoids overlap.
